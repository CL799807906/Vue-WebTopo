技术栈：

1、框架依托于若依框架，符合中小企业的开发需求。

2、微信小程序、安卓APP是基于uniapp开发的。

3、2D云组态是基于vue开发的，内嵌elementUI，符合大众开发习惯。

6、数据库存储使用mysql和taos时序数据库，支持亿级数据分析。
